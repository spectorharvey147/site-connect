# ClaimFlow Pro Docs

This folder contains the current handover and operating guides for the project.

Files:

- `USER_GUIDE.md`
  End-user guide for login, claim submission, approval, history, vouchers, and settings.
- `SUPABASE_MIGRATION_AND_SETUP_GUIDE.md`
  Full guide to move the app to a new Supabase account/project, including schema, storage, RLS, secrets, and checks.
- `EMAIL_AND_GMAIL_SETUP_GUIDE.md`
  How to replace the Gmail sender, configure SMTP secrets, and validate email notifications.
- `DEPLOYMENT_AND_RELEASE_GUIDE.md`
  Web deployment, Capacitor sync, Android release build, and operational checklist.
